# Highest-Grossing Tours by Women - Data Cleaning

Este projeto explora um dataset baseado no **Top 20 highest-grossing concert tours by women of all time**.

## Objetivos do Projeto

- Limpar e pré-processar dados brutos extraídos da Wikipedia
- Tratar valores ausentes, duplicatas, formatos inconsistentes e tipos de dados

## Fonte dos Dados

Wikipedia: Lista de turnês femininas de maior bilheteria.

## Principais Etapas de Limpeza

- Padronização dos nomes das colunas
- Remoção de caracteres especiais e referências
- Conversão de colunas monetárias para valores numéricos
- Tratamento de valores ausentes com validação na fonte original
- Remoção de colunas irrelevantes
- Correção de inconsistências no ranking

## Ferramentas Utilizadas

- Python
- Pandas
- Jupyter Notebook
